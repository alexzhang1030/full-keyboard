# obsidian 4

双链

## 快捷键

### 添加内部链接

- `[[`: 添加链接
- `[[#`: 添加链接具体指定到某个标题
- `[[|`: 通过 `|` 来修改在 reading view 中的名字
- `cmd + mouse_hover`: 预览笔记(需要手动将这个功能打开)

### 嵌入

- `![[`: 嵌入笔记(支持视频)
- 通过 `insert attachment` 来导入视频

### 新的窗格打开链接

预览模式：

- `cmd + mouse_click`

编辑模式：

- `cmd + shift + mouse_click`
- `cmd + opt + enter`

### 打开链接

- `opt + enter`

### note refactor

这是一个插件，提供了两个功能：

- `cmd + shift + c`: 选中的内容添加到笔记中
- `cmd + shift + n`: 选中的内容新建笔记
