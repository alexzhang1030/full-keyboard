# obsidian 2

文件操作的常见指令

## 快捷键

- `cmd + n`: 创建
- `cmd + w`: 关闭
- `cmd + shift + l`: 回复关闭的文件
- `cmd + p`: 检索最近打开的文件
- 删除文件可以配一下指令：
  - `exmap deleteFile obcommand app:delete-file`
  - `nmap df :deleteFile`
- 分屏：`split horizontal` 以及 `split vertically` 需要手动配置
  - 推荐可以使用 `cmd + \` 以及 `cmd + shift + \`
- `shift + 方向键` 在屏幕之间移动(需要手动配置，指令以 `Focus` 开头)
- `cmd + shift + p`: 命令面板
