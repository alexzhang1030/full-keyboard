# chrome 6

快捷键操作 DevTools

## 快捷键

- 打开 DevTools
  - `option + command + i` 这个会打开你上次浏览的面板
  - `option + command + j` 打开 devTools 并聚焦 `console`
  - `shift + command + c` 打开 devTools 并聚焦 `elements` 面板
- `command + p` 打开查找文件面板
- `shift + command + p` 打开指令面板
- ctrl + ` 聚焦 console 面板
- `command + k` 清空 console
- `command + [|]` 切换面板

值得注意的是，Chrome 的快捷键内置了 VSCode 的预设
