# vscode 4

vscode 高效编码技巧

## 一些常用的快捷键

- `command + .`: 展示 code actions
- `shift + command + space`: 触发函数的参数提示
- `command + i`: 触发建议
- `opt + up/down`: 快速移动行
- `command + enter`: 增加行
- `opt + delete`: 删除光标前面的单词
- `opt + ctrl + delete`: 基于单词删除光标前面的单词，驼峰命名法会被视为多个单词
- `f8`: 跳转到下一个错误处
- `shift + f8`: 跳转到上一个错误处
- `command + f2`: 选择当前文件中所有当前光标指向的单词
