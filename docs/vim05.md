# vim5

掌握可视化模式

## 可视化

- `V`: `--VISUAL LINE--` 可视化行模式
- `v`: `--VISUAL--` 可视化字符模式，以单个字符作为单位
- `ctrl + v`: 可视化块模式

## 退出

- `esc`: 退出可视化模式
- `ctrl + [`: 退出可视化模式
- `v`/`V`

## 导航

- `o`: 在可视化模式下修改光标的位置
- `gv`: 回到上次可视化模式的选择位置

## 技巧

- 跨多行编辑，例如给多行加上分号，配合上 `A` 或者 `I`
- 尽量少用可视化模式，因为会增加操作的步骤
- 赋值粘贴，利用剪切板
